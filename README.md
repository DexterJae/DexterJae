## 🧠 Research

I am currently involved in research projects related to:

### LLM Reasoning & KV Cache Compression

Researching how KV cache compression affects reasoning performance in large language models.
My work includes evaluating compression methods, analyzing accuracy-latency tradeoffs, and optimizing decoding-time compression strategies.

### Embodied Multimodal Navigation

Working on embodied agent tasks involving 3D environments, navigation, visual perception, and question answering.
I have experience with simulation environments such as **Isaac Sim**, **AI2-THOR**, and **MP3D**.

### Multimodal Reasoning Benchmarks

Contributing to benchmark development for evaluating AI reasoning across spatial, temporal, physical, mathematical, abstract, and planning dimensions.

### Bioinspired Robotics

Previously worked on a flapping-wing robot project focused on silent flight and fire detection using computer vision models.
